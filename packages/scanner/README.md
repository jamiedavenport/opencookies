# @opencookies/scanner

Static cookie and vendor detection for OpenCookies. Scans built output and source for known third-party scripts and the cookies they set.

> Status: scaffold. Detection rules land in [OP-308](https://linear.app/open-policy/issue/OP-308).

## Install

```sh
bun add -D @opencookies/scanner
```
