# @opencookies/svelte

Svelte adapter for OpenCookies. Exposes a readable store interface (works with `$store` in Svelte 4 and `$state` wrapping in Svelte 5).

> Status: scaffold.

## Install

```sh
bun add @opencookies/core @opencookies/svelte
```

## Peer dependencies

- `svelte >= 4`
