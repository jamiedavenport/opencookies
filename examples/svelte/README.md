# @opencookies/example-svelte

Svelte example app for OpenCookies. Demonstrates wiring `@opencookies/svelte` into a Vite SPA: a
banner, preferences modal, per-category checkboxes, and a `<ConsentGate>` around analytics
content.

## Develop

```sh
vp run example-svelte#dev
```

## Build

```sh
vp run example-svelte#build
```
