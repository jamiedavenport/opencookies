# @opencookies/example-solid

Solid example app for OpenCookies. Demonstrates wiring `@opencookies/solid` into a Vite SPA: a
banner, preferences modal, per-category checkboxes, and a `<ConsentGate>` around analytics
content.

## Develop

```sh
vp run example-solid#dev
```

## Build

```sh
vp run example-solid#build
```
