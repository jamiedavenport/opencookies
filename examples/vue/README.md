# @opencookies/example-vue

Vue example app for OpenCookies. Demonstrates wiring `@opencookies/vue` into a Vite SPA: a
banner, preferences modal, per-category checkboxes, and a `<ConsentGate>` around analytics
content.

## Develop

```sh
vp run example-vue#dev
```

## Build

```sh
vp run example-vue#build
```
